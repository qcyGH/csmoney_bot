# csmoney_bot
Данный телеграм-бот парсит сайт cs.money. Он поможет собрать оружие с определённой категории, при этом фильтрует по скидке (по умолчанию она равняется >= 10), чего нельзя сделать на сайте. Так же, для удобства, при подборе скинов, включены фильтры: "Качество: Прямо с завода", "Скины без блока".
# Бот
Для бота сделано меню и информативные сообщения.
